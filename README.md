# E-Commerce Product Scraper

## Overview

This project is a Java-based web scraping application developed as part of the SkillCraft Technology Software Development Internship.

The application extracts product information such as product names, prices, and ratings from an e-commerce website and stores the collected data in a structured CSV file.

## Features

- Extracts product names
- Extracts product prices
- Extracts product ratings
- Stores data in CSV format
- Uses Jsoup for HTML parsing
- Handles large amounts of product data efficiently

## Technologies Used

- Java
- Jsoup
- Maven
- CSV File Handling

## Website Used

https://books.toscrape.com

## Output

The program generates a CSV file named:

products.csv

The file contains:

| Name | Price | Rating |
|------|--------|---------|
| Product Name | Product Price | Product Rating |

Successfully scraped and stored 1000 book records.

## How to Run

1. Clone the repository
2. Install Maven dependencies
3. Run Main.java
4. Check products.csv for the extracted data

## Learning Outcomes

- Web Scraping using Java
- HTML Parsing with Jsoup
- File Handling in Java
- Data Extraction and Storage
- Maven Dependency Management
